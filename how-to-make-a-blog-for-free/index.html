How to make a blog for free

02 April 2015

I made this blog for free, and so can you. I created it using the Ghost framework and hosted it through my GitHub pages, then added free Disqus comments. Here are the steps I took and some links to the various instructions I read.

My main source of helpful instructions from LED Technica. Before I direct you there, make sure that you have installed the following:

  1. Python (from this site)
  2. Node.js (available here)

If you don't remember whether you have installed them, type into your terminal:

$ which python

This returns "/usr/bin/python", or some similar file path. Same for Node.js:

$ which node
>>/usr/local/bin/node

Also make sure that your Python includes pip, a Python package manager:

$ which pip
>>/usr/local/bin/pip

Install pip if you do not have it (here's how).

Once you have installed each of the above, you are ready to walk through LED Technica's instructions. The basic process is:

  1. Run a Ghost blog with node.js locally (i.e., using my computer as a server so only I can see it).
  2. Use a Python package called Buster (as in GhostBusters) which will read the static files off my local server, and tidy them up for a push to GitHub. This step will require you to install Python and Pip .

After finishing LED Technica's instructions, you still need to go to your 'ghost-0.5.8' (or 'ghost-#.#.#') folder and edit the 'config.js'. Find the 'config.production.url' and 'config.development.url' and replace them with your blog's url (such as 'http://sdwebster.github.io/' or 'http://irruncibly.so', then restart your local server before calling Buster again. If you don't do this, your "Home" button will redirect to localhost:2368, and your blog will break.

Later on, I added Disqus comments (also free of charge) by following these other instructions. I had to dig around a bit when they started talking about my “post.hbs” file - it turned out to be at ".../ghost-0.5.8/content/themes/casper/post.hbs".

At this point, my new blog was online at http://sdwebster.github.io, entirely for free. The only drawback to doing it this way is that on GitHub pages, I can only serve static files, and can't demo the projects I've worked on.

Finally, I bought the Somalian domain name "irruncibly.so". I linked it to my GitHub pages following these clear instructions, so that now http://sdwebster.github.io redirects to irruncibly.so. I was confused at first when the instructions referred to "192.30.252.153", then I realized that it's simply an IP address for GitHub.

Happy blogging!
